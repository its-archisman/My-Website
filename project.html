  • Image-to-LaTeX Formula Translation Using Encoder-Decoder Architecture : Prof. Parag Singla [November 2023]
    • Developed an Encoder-Decoder model to translate images of mathematical expressions into LaTeX code, utilizing CNN as encoder and RNN as decoder, along with an LSTM layer
    • Implemented a CNN-based encoder with multiple convolutional and pooling layers to extract context vectors from resized and normalized images
    • Trained the model on a synthetic dataset and fine-tuned it on a handwritten dataset, reporting BLEU scores on validation and test sets to measure accuracy
    • Used teacher forcing for effective LSTM training, incorporating learnable embeddings and padding for handling variable-length output sequences

    • Tested the impact of using ReLU activation functions compared to sigmoid, and experimented with adaptive learning rates for improved training efficiency
    • Used scikit-learn's MLPClassifier for comparison, replicating the custom neural network architecture and evaluating the differences in performance and training time
